# Avro to PHP

:construction: This is a work-in-progress! :construction:

Compile Avro .avsc files into usable PHP classes.

### Installation
```bash
$ composer require chasdevs/avro-to-php
```

### Usage
```bash
$ vendor/bin/avro-to-php compile:directory dir/with/avsc/
```
